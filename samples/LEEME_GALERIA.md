# Galería Quantum v5.0 (Dinámica)

Esta carpeta contiene una selección de muestras de alta fidelidad para alimentar el simulador del Dashboard sin coste de API.

- **Variedad Estelar:** He incluido 3 estilos diferentes para cada serie (**Faces** y **Abstract**).
- **Lógica Inteligente:** El Dashboard ahora seleccionará una de estas 6 imágenes al azar cada vez que generes un nuevo prompt, para que no veas siempre la misma foto.

### Contenido de la carpeta:
- `face_v5_1.png` a `face_v5_3.png`: Retratos cibernéticos variados.
- `abstract_v5_1.png` a `abstract_v5_3.png`: Conceptos abstractos de energía.
